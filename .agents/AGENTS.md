# Financial Diary Project - Developer Ideas & Requirements

## Core Features & Menu Structure
Aplikasi pencatatan finansial sehari-hari dengan struktur menu komprehensif:

### 1. Dashboard (Beranda)
- Ringkasan Saldo (Total uang di semua dompet/rekening).
- Highlight Bulan Ini (Pemasukan vs Pengeluaran bulan berjalan).
- Shortcut cepat untuk tambah transaksi.

### 2. Transaksi (Transactions)
- **Pemasukan (Income)**: Pencatatan pemasukan harian.
- **Pengeluaran (Outcome)**: Pencatatan pengeluaran harian.
- **Transfer Antar Akun**: Pencatatan perpindahan uang antar dompet/rekening.

### 3. Perencanaan & Target (Planning & Goals)
- **Anggaran (Budgets)**: Fitur untuk membatasi pengeluaran bulanan per kategori.
- **Tujuan Keuangan (Financial Goals)**: Fitur menabung untuk target tertentu dengan visual progress bar.

### 4. Manajemen Akun & Aset (Accounts & Assets)
- **Dompet / Rekening (Wallets)**: Mencatat posisi saldo di berbagai tempat (Kas Tunai, Rekening Bank, e-Wallet).
- **Pencatatan Aset**: Memantau benda bernilai atau investasi jangka panjang (Emas, Reksadana, Kendaraan).
- **Hutang & Piutang (Debts & Loans)**: Mencatat riwayat pinjaman uang beserta tanggal jatuh temponya.

### 5. Laporan & Analitik (Reports & Analytics)
- **Summary Keuangan**: Ringkasan finansial per bulan dan tahunan.
- **Grafik Chart**: Visualisasi data berupa grafik (contoh: Pie chart untuk kategori pengeluaran, Bar chart untuk tren bulanan).
- **Kalender Keuangan**: Tampilan kalender untuk melihat aktivitas keluar-masuk uang berdasarkan tanggal.

### 6. Master Data
- **Kategori**: Manajemen daftar kategori untuk pemasukan dan pengeluaran.

### 7. Pengaturan (Settings)
- **Setting User**: Pengaturan informasi akun dan profil pengguna.
- **Ekspor / Impor Data**: Fitur untuk men-download riwayat data ke format Excel (CSV) atau PDF.

### 8. AI Assistant (Hermes Agent)
- **Chatbot Keuangan**: Integrasi API dengan agent AI "Hermes" di VPS untuk interaksi via chat. Pengguna bisa mencatat transaksi hanya dengan mengetik kalimat (contoh: "Saya baru saja beli kopi 30 ribu" -> otomatis masuk ke pengeluaran).
- **Saran & Analisis Otomatis**: AI membaca data keuangan (summary) untuk memberikan saran berhemat, mendeteksi pola pemborosan, atau memberi peringatan saat batas anggaran hampir habis.
